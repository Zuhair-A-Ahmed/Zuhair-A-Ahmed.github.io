# Deployment

The live site is expected to deploy from the `master` branch through GitHub Pages.

Before merging a website pull request:

1. Review the full diff.
2. Preview `index.html` locally.
3. Check navigation, email, LinkedIn, GitHub, and project links.
4. Check mobile responsiveness.
5. Confirm that no customer-confidential or unnecessary personal information is published.

After merging, verify the production website and GitHub Pages deployment status.
