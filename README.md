# Zuhair A. Ahmed — Portfolio

Personal portfolio website for Zuhair A. Ahmed, focused on SCADA, DCS, SIS, industrial automation, electrical substations, commissioning, and robotics research.

## Website

The production website is hosted with GitHub Pages from the `master` branch.

## Structure

- `index.html` — semantic page structure, portfolio content, and responsive CSS
- No external JavaScript or build process is required

## Local preview

Open `index.html` directly in a browser, or run a simple local server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

Changes should be reviewed through a pull request before merging into `master`. GitHub Pages will publish the merged version automatically according to the repository Pages settings.
